rlevinson

This function computes the autocorrelation coefficients using prediction polynomial method.

Calling Sequence

R = rlevinson(a, efinal)
[R, U] = rlevinson(a, efinal)
[R, U, kr] = rlevinson(a, efinal)
[R, U, kr, e] = rlevinson(a, efinal)

Parameters

a:

Input argument ,prediction polynomial coefficients.

efinal:

Input argument, final prediction error.

R:

Returns the auto-correlation coefficients.

U:

Returns upper triangular matrix of order length(a)*length(a).

kr:

Reflection coefficients.

e:

Prediction error.
