sosbreak

function for breaking a polynomial in second order polynomials (and an extra linear)

Calling Sequence

[zerosort,g]=sosbreak(p)

Parameters

p:

-the input polynomial.

zerosort:

-the array of the broken polynomials

g:

the interger multiple obtained after breaking the polynomial

Examples

v=[1+4*%s+6*%s^2+4*%s^3+%s^4];
[zerosort,g]=sosbreak(v);
//OUTPUT:
// g  =1.  
 //zerosort  =
 //   zerosort(1)
//           2
// 1 + 2s + s   
// zerosort(2)
//                  2 
//1.0000000 + 2s + s
