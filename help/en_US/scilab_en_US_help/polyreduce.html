polyreduce

Calling Sequence

p= polyreduce (c)

Parameters

c:

polynomial coefficients vector.

p:

Reduced polynomial coefficients.

Description

Reduce a polynomial coefficient vector to a minimum number of terms by stripping off any leading zeros. This function is used in residue().

Examples

polyreduce ([0, 0, 1, 2, 3])
//Output: [1, 2, 3]
