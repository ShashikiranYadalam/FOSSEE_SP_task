zerocrossing

This function estimates the points at which a given waveform y=y(x) crosses the x-axis using linear interpolation.

Calling Sequence

w = zerocrossing (x, y)

Parameters

w:

vector of points at which the function y(x) crosses x-axis.

y:

The dependant variable,y(x)

x:

The independant variable

Description

This is an Octave function. This function estimates the points at which a given waveform y = y(w) crosses the x-axis. It uses linear interpolation.

Examples

x = linspace(0,1,100);
y = 2*sin(2*%pi*x);
x0= zerocrossing(x,y);
//ans:
// x0  =  0.    0.5  
y0=interp1(x,y,x0)
plot(x,y,x0,y0,'x')
